## Required

- Node.js >= 4

## Getting started

```shell
$ git clone git@github.com:kamataryo/qitta.git
$ cd qitta
$ npm install
```

Start a dev-http-server:

```shell
$ npm start
```

Then build:

```shell
$ npm run build
```

## Automated testing

```shell
$ npm test
```
